# 计算机四级数据库等级考试
## 功能
1.一共30套题，包含操作系统和数据库的题库。其中操作系统和数据库都是30道单选，10道多选
2.错题和正确题目记录，方便做完回头重写做错题
3.题目解析，每个题都有解析
